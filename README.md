chatgpt-telegram-bot

## Install

```
go install github.com/pefish/chatgpt-telegram-bot/cmd/chatgpt-telegram-bot@latest
```

## Quick start

```shell script
chatgpt-telegram-bot --config=/path/to/config
```

or

```shell script
GO_CONFIG=/path/to/config chatgpt-telegram-bot
```
